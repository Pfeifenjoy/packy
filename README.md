# Packy

Packy is a 2D game for educational purposes.


## Prerequisites

- [poetry](https://python-poetry.org/)

## Build

Run

```
poetry install
poetry build
poetry run packy
```


```
python -m pip install pygame pydantic
python -m packy
```

To install from vs code: go to install_requirements.py
To run the game: go to run_game.py
